The HTTP Process

HTTP works through a request and response process. The browser acts as the client. The web server stores the website files and sends them back when requested.

HTTP Request and Response

This diagram shows an example of a server.

Client

The client is usually a web browser. It sends a request for a web page or resource.

Server

The server receives the request and sends back a response with the requested content.

Simple Steps

  1. User enters a website address.
  2. The browser sends an HTTP request.
  3. The server processes the request.
  4. The server sends an HTTP response.
  5. The browser displays the webpage.
