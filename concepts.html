Important HTTP Concepts

HTTP

Hypertext Transfer Protocol is the standard protocol used for communication between web browsers and web servers.

HTTPS

HTTPS is the secure version of HTTP. It encrypts information using TLS to protect user data.

Client

The client is usually a web browser that requests information from a server.

Server

A server stores websites and sends requested resources back to the client.

Request

A request is sent by the browser asking the server for a webpage or another resource.

Response

A response is the information returned by the server after processing a request.
